What's HTML?

L'HyperText Markup Language, généralement abrégé HTML, est le langage de balisage conçu pour représenter les pages web. C'est un langage permettant d'écrire de l'hypertexte,d'où son nom

CSS

Les CSS, Cascading Style Sheets (feuilles de styles en cascade), servent à mettre en forme des documents web, type page HTML ou XML

JS

JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives mais aussi pour les serveurs2 avec l'utilisation (par exemple) de Node.js3.

Bootstrap

Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

Adresse IP vs adresse MAC

MAC signifie Media Acess Control alors que IP signifie Internet Protocol. ... D'autre part, l'adresse MAC identifie un périphérique participant à un réseau.L'adresse MAC est une adresse hexadécimale de 48 bits(6 octets) alors que l'adresse IP a deux versions, IPv4 est une adresse 32 bits et IPv6 est une adresse 128 bits

open source vs proprietary

Open Source vs. Proprietary Software. The term open source refers to software whose source code — the medium in which programmers create and modify software — is freely available on the Internet; by contrast, the source code for proprietary commercial software is usually a closely guarded secret.
